alert(_:​dismiss:​) 

@ViewBuilder public func alert<Action>(
    _ store: Store<AlertState<Action>?, Action>,
    dismiss: Action
  ) -> some View

Displays an alert when then store's state becomes non-nil, and dismisses it when it becomes nil.

Parameters

Name Type Description
store Store<Alert​State<Action>?, Action>

A store that describes if the alert is shown or dismissed.
dismissal

An action to send when the alert is dismissed through non-user actions, such as when an alert is automatically dismissed by the system. Use this action to nil out the associated alert state.
